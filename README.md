# blockBOUNTY
blockchain bugbounty framework

## INSTALLATION

Clone the repository

```bash
git clone https://github.com/ekomsSavior/blockBOUNTY.git
cd blockBOUNTY
```

Install required tools and libraries

```bash
sudo apt update
pip install requests.txt 
```

## RUNNING THE TOOL


launch the tool

